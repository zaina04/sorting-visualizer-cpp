# Sorting Visualizer (C++)

This is a menu-driven sorting visualizer implemented in C++.

## Features
- Bubble Sort
- Selection Sort
- Step-by-step visualization of sorting process
- Menu-driven interface

## How to Run
1. Compile the code:
   g++ sorting.cpp -o sorting
2. Run:
   ./sorting

## Concepts Used
- Data Structures (Arrays)
- Sorting Algorithms
- Time Complexity
- Structured Programming
